Проект: Анализ продаж Amazon India (2022)
Инструменты: SQL (SQLite), Excel

Что сделано:
- 5 SQL-запросов: топ категорий, динамика по месяцам, 
  средний чек (JOIN двух таблиц), топ городов и регионов
- Excel-дашборд с 5 графиками

Ключевые выводы:
- Категория Set лидирует по выручке (39 млн INR), 
  Kurta — по количеству заказов (46 тыс.)
- Выручка падает во втором полугодии: пик в феврале (2.98 млн), 
  минимум в ноябре (2.55 млн)
- Средний чек Set вдвое выше чем у Kurta (834 vs 449 INR)
- Бенгалуру — главный рынок сбыта (10 685 заказов)
- Махараштра генерирует наибольшую выручку среди регионов

Данные: Amazon Sales Dataset
Источник: https://www.kaggle.com/datasets/thedevastator/unlock-profits-with-e-commerce-sales-data
